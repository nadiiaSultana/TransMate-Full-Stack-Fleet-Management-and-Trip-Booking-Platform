# Project Features

## Authentication

- Customer registration
- Driver registration
- JWT login
- Role-based access
- Admin driver approval
- User suspend/activate

## Vehicle Management

- Vehicle type CRUD
- Vehicle CRUD
- Driver-to-vehicle assignment
- Vehicle availability tracking
- Maintenance status tracking

## Booking Management

- Customer booking creation
- OpenStreetMap route selection
- Automatic distance calculation
- Automatic fare estimation
- Booking cancellation
- Admin booking acceptance

## Trip Management

- Admin trip assignment
- Driver accepts trip
- Driver starts trip
- Driver completes trip
- Vehicle status auto-update
- Booking status auto-update

## Payment

- Cash/mock payment
- bKash sandbox-ready structure
- Nagad sandbox placeholder structure
- Payment confirmation
- Refund status
- PDF invoice generation

## Notifications

- Database notifications
- Console email notifications by default
- Trip assigned email
- Trip started email
- Trip completed email

## Reports

- Admin dashboard summary
- Booking report
- Payment report
- Vehicle usage report
- Driver earnings report
- Maintenance report

## Maintenance

- Schedule vehicle maintenance
- Mark maintenance in progress/completed
- Track maintenance cost
- Auto-set vehicle status to MAINTENANCE
- Auto-restore vehicle to AVAILABLE after completion
